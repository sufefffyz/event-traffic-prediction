# Reproduction Code

This directory contains reference implementations for event-based traffic forecasting papers that are useful for reproduction, comparison, and method inspection.

These projects are not the primary framework for this repository. New research development should happen around `../BasicTS/`, while this directory should stay close to the imported upstream layouts unless a reproduction fix is necessary.

## Contents

| Directory | Purpose |
| --- | --- |
| `ConFormer/` | Reproduce and inspect ConFormer. |
| `IGSTGNN/` | Reproduce and inspect IGSTGNN. |
